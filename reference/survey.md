# Contact survey (defunct)

**\[defunct\]**

`survey()` is defunct. Use
[`as_contact_survey()`](https://epiforecasts.io/socialmixr/reference/as_contact_survey.md)
instead.

## Usage

``` r
survey(participants, contacts, reference = NULL)
```

## Arguments

- participants:

  a `data.frame` containing information on participants

- contacts:

  a `data.frame` containing information on contacts

- reference:

  a `list` containing information needed to reference the survey, in
  particular it can contain a "title", "bibtype", "author", "doi",
  "publisher", "note", "year"

## Value

Always errors.

## See also

[`as_contact_survey()`](https://epiforecasts.io/socialmixr/reference/as_contact_survey.md)
